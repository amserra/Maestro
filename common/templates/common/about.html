What is Maestro?

  • Maestro is a system that can help you get the data that you need. It can gather and classify data according to the users' preferences, which are called search contexts. A user can create multiple search contexts, and search contexts can be shared among users through organizations.
  • Maestro gathers data through gather plugins. These are programs that can fetch data from different sources: crawling websites, through APIs, RSS feeds, or others. Maestro's architecture makes it easy to create new gather plugins, by providing developers an easy, pluggable interface.
  • The same plugin-driven approach is taken on the classification part. Classification programs can be as simple as a set of conditional statements, and as complex as an ML model. Developers can encapsulate their classification programs in a simple interface that can be integrated with Maestro easily. Note that a user may opt to use Maestro only to gather data.
  • Finally, by defining a destination target through a webhook, your program can receive and use the data gathered and classified by Maestro to your end-purpose.
